Steps to run the code:
  1.  Clone the repo
  2.  Optinal:
      a. create a venv using: python -m venv venv
      b. activate the venv: venv\Scripts\activate
  3.  pip install -r requirements.txt
  4.  python app.py
  5.  go to http://127.0.0.1:5000
